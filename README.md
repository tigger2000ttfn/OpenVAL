## Design System

PHAROLON has a purpose-built design language вҖ” dark, premium, and instantly distinctive.

- **Gold `#E2A837`** primary identity вҖ” the lighthouse gold, not another tech teal
- **Dark Teal `#00A090`** secondary вҖ” validated systems, quality assurance
- **Cormorant Garamond** display typeface вҖ” editorial weight for a platform that takes safety seriously
- **Always-dark chrome** вҖ” charcoal `#141414` header + dark teal sidebar in both themes
- **Warm pass/fail** вҖ” lime `#86C140` for pass, coral `#E54B2E` for fail
- **`#c1c1c1` light background** вҖ” white cards on medium gray, premium not sterile
- **Modal-first** вҖ” every deliberate action uses a modal. Toasts only for background events.
- **Collapsible sidebar** вҖ” 192px expanded labels, 52px collapsed icon-only with tooltips
- **SOP Visualizer** вҖ” pharma's first AI-powered interactive process flowchart

## 21 CFR Part 11 Compliance Architecture

| Requirement | Implementation |
|---|---|
| Audit trails `11.10(e)` | `before_flush` intercepts every write. SHA-256 chain. Nightly Celery verification. |
| Electronic signatures `11.50` | Re-authentication at every signing. Server-side timestamp only. Content hash linked at signing. |
| Access controls `11.10(d)` | RBAC, unique user IDs, concurrent session limits, periodic access review workflow. |
| System validation `11.10(a)` | Ships with validation package VP-001 through VP-015. |
| ALCOA+ | Attributable В· Contemporaneous В· Original В· Enduring В· Available вҖ” enforced by architecture. |
| FDA CSA 2025 | Dual-mode CSV/CSA. Intended use per feature. Unscripted testing capture. |
| EU GMP Annex 1 (2022) | CCS, APS, PUPSIT document types. Continuous EM. RABS/isolator protocols. |
| QMSR (Feb 2026) | Design History File, design input/output traceability, ISO 13485 management review. |

## Quick Start

```bash
# Requires Ubuntu 22.04+ or RHEL 9 В· PostgreSQL 15+ В· Redis 7+ В· Node.js 20+ В· Python 3.11+
git clone https://github.com/tigger2000ttfn/PHAROLON.git pharolon

```

**Development:**

```bash
# Terminal 1 вҖ” backend with hot reload
cd backend && python -m venv venv && source venv/bin/activate
pip install -r requirements-dev.txt
uvicorn app.main:app --reload --port 8000
# вҶ’ Swagger docs: http://localhost:8000/api/docs

# Terminal 2 вҖ” frontend with HMR
```
